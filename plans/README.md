## Verification Baseline

The repo's canonical gate is `make verify`, which runs format check, lint/staticcheck/vet, tests, race tests, security checks, coverage, build, and smoke. During the latest audit, `go test ./...`, `go test -race ./...`, and `go test -cover ./...` passed. `make verify` was not run by the advisor because it mutates local build artifacts by design.
